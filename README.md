# Online Examination System

## Objective
The objective of this project is to develop an Online Examination System that allows users to securely participate in online tests, answer multiple-choice questions (MCQs), manage their profiles, and receive results efficiently. The system aims to automate the examination process while ensuring accuracy and convenience.

## Steps Performed
1. Designed the user authentication module for secure login.
2. Implemented profile and password update functionality.
3. Created an interface for displaying and answering MCQs.
4. Added a timer feature to monitor exam duration.
5. Implemented automatic submission when the time expires.
6. Developed session management and logout functionality.
7. Tested the system to ensure smooth exam execution and result processing.

## Tools & Technologies Used
- Java
- Swing/AWT (for GUI, if applicable)
- MySQL Database (if applicable)
- JDBC Connectivity
- VS Code / Eclipse / IntelliJ IDEA
- Git & GitHub

## Outcome
Successfully developed an Online Examination System that enables secure login, profile management, MCQ-based assessments, timer-controlled exams, and automatic submission. The system streamlines the examination process and provides a reliable platform for conducting online tests.
